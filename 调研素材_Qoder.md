## 一、产品概述

Qoder是阿里巴巴推出的面向全球的**Agentic编程平台**，2025年8月正式发布，是阿里AI编程产品线的国际化布局。

- **定位**：Agentic编程平台
- **目标市场**：全球开发者
- **与通义灵码关系**：Qoder面向海外，通义灵码面向国内

## 三、核心功能

### 3.1 Agent Mode（核心功能）

- 用户描述需求
- 自动分析项目结构
- 自动查找相关文件
- 自动生成代码
- 自动修复Bug
- **特点**：全自动化的Agent编程体验

### 3.2 记忆感知

- **业内唯一** JetBrains原生记忆感知插件
- 自动形成个人编码记忆
- 类似通义灵码的自动记忆，但深度集成JetBrains

### 3.3 Quest模式（Spec First）

- "Spec First"需求先行理念
- 先整理Spec文档再执行开发
- 类似CodeBuddy的Builder模式
- **差异**：更强调文档规范

### 3.4 Inline Chat

- 选中代码即时对话
- 支持重构/优化/解释
- 类似CodeBuddy的行间会话

### 3.5 智能代码补全

- AI补全建议
- 体验类似GitHub Copilot
- 支持多语言

### 3.6 多模型自动路由

- 接入Claude、GPT、Gemini等多个模型
- 自动匹配最优模型
- **特点**：用户无需手动选择模型

### 3.7 10万文件级理解

- 云端代码图谱+本地索引
- 无需手动添加上下文
- 支持超大型项目

## 四、定价

| 版本 | 价格 | 说明 |
|------|------|------|
| 免费试用 | 免费 | 有限额度 |
| 付费版 | **2美元/月**（首购）→ 10美元/月（续费） | 无限代码补全，多模型使用 |

**分析**：
- 首购价格极低（2美元/月），吸引试用
- 续费价格为10美元/月，在国际市场有竞争力
- 相比Cursor的20美元/月更便宜

## 五、与通义灵码的定位差异

| 维度 | 通义灵码 | Qoder |
|------|----------|-------|
| 目标市场 | 国内 | 全球 |
| IDE支持 | VS Code + JetBrains | 主要JetBrains |
| 底层模型 | Qwen系列 | Claude/GPT/Gemini（多模型） |
| 核心特色 | 自动记忆、NES行间预测 | Agent Mode、10万文件理解 |
| 定价 | 人民币（79~159元/月企业版） | 美元（2~10美元/月） |
| 语言 | 中文为主 | 英文为主 |

**问题**：阿里4条AI编程产品线（通义灵码、Qoder、Qwen Code、通义灵码AI IDE）品牌碎片化严重，用户选择困惑。
