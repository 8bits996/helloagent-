## 🚀 使用步骤

### 第一步：准备阶段（1月26日15:00前完成）

1. **登录12306账号**
   - 访问 https://www.12306.cn/
   - 登录你的12306账号

2. **添加乘车人**
   - 进入"我的" → "常用联系人"
   - 确保以下4位乘客都已添加：
     * 陈卓
     * 冯巧玉
     * 陈妙可
     * 陈苇杭

3. **进入购票页面**
   - 设置出发站：襄阳
   - 设置到达站：武当山西
   - 设置日期：2026-02-09
   - 点击"查询"

### 第二步：准备脚本（15:30左右）

1. 打开浏览器开发者工具
   - Windows: 按 `F12` 键
   - 或右键 → 检查

2. 切换到 Console 标签页

3. 打开脚本文件
   - 在本目录找到 `xiangyang_wudangshan_grab.js` 文件
   - 用记事本或其他编辑器打开
   - 全选复制（Ctrl+A, Ctrl+C）

4. 粘贴到浏览器控制台
   - 回到12306页面的Console标签
   - 粘贴脚本（Ctrl+V）
   - 按回车执行

5. 确认加载成功
   - 控制台应显示绿色边框的使用说明
   - 最后显示 "✓ 脚本已加载，弹窗监控已启动"

### 第三步：定时抢票（15:50左右）

在控制台执行以下命令：

```javascript
GrabXY.schedule()
```

脚本将：
- 显示倒计时（距离16:00还有XX秒）
- 在16:00前300ms自动开始抢票
- 自动查找有票车次
- 自动点击预订
- 自动选择4位乘客
- 自动修正陈苇杭为儿童票
- 自动提交订单

### 第四步：支付

- 如果抢票成功，页面会跳转到支付页面
- 在15分钟内完成支付

## 🎮 常用命令

在浏览器控制台可以使用以下命令：

| 命令 | 说明 |
|------|------|
| `GrabXY.schedule()` | 定时16:00自动抢票（推荐） |
| `GrabXY.go()` | 立即开始抢票（手动触发） |
| `GrabXY.stop()` | 停止抢票 |
| `GrabXY.order()` | 手动处理订单页并提交 |
| `GrabXY.test()` | 测试订单页（不提交，仅验证乘客和票种） |

## ⚠️ 重要提示

1. **提前添加乘客**
   - 4位乘客必须在开售前添加到常用联系人
   - 否则无法自动选择

2. **保持页面活跃**
   - 抢票过程中不要切换到其他标签页
   - 不要关闭浏览器
   - 不要执行刷新操作

3. **网络要求**
   - 确保网络稳定
   - 建议使用有线网络或5G WiFi
   - 避免使用信号差的网络

4. **时间同步**
   - 确保电脑时间准确
   - 可以与手机时间对比校准

5. **备用方案**
   - 如果主站（襄阳）没票，可以尝试"襄阳东"
   - 如果直达没票，可以考虑中转方案

## 🔧 故障排除

### 问题1：控制台没有显示加载成功
**解决**：
- 确保是在12306的购票页面（查询结果页面）
- 刷新页面重新粘贴脚本

### 问题2：显示"未找到乘客列表"
**解决**：
- 确保乘客已添加到常用联系人
- 刷新页面重试

### 问题3：自动提交失败
**解决**：
- 手动点击提交按钮
- 或执行 `GrabXY.order()` 命令

### 问题4：脚本停止运行
**解决**：
- 执行 `GrabXY.go()` 重新开始抢票
- 或执行 `GrabXY.schedule()` 重新定时
