## 🆘 遇到问题?

### 常见问题

**Q: Server 启动失败?**
A: 检查是否在代码中打印到 stdout,使用 stderr 代替

**Q: 工具调用失败?**
A: 检查参数类型是否匹配,查看错误信息

**Q: 连接超时?**
A: 检查 Server 是否正确启动,查看进程是否存在
