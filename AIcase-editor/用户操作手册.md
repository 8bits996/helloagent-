# 我使用司内工具CodeBuddy调教了一个 AI 帮你做案例：2小时 → 10分钟

以前做一个案例：整理数据、找亮点、写文案、排版设计… 2小时起步, 现在：把数据丢给 AI，10 分钟出成品(初稿)

## AI案例生成器能帮你什么？

你只需要： 把投放数据表格粘贴给 AI

AI 帮你：

● ✅ 自动分析数据，找出最有说服力的亮点

● ✅ 自动撰写案例文档，专业表述不用愁

● ✅ 自动生成精美 HTML，横版竖版一键出图

● ✅ 智能脱敏处理，该藏的数据帮你藏

最终产出： 可以直接发给客户/发朋友圈的案例长图

## ⏱️ 3分钟上手

### 第一步 CodeBuddy安装，下载案例生成器；

1、CodeBuddy下载链接：[腾讯云代码助手 CodeBuddy - AI 时代的智能编程伙伴](https://copilot.tencent.com/ide/)

2、登录时选择 IOA登录：

![](https://wdoc-76491.picgzc.qpic.cn/MTY4ODg1MDUyOTQ5ODE3Nw_39200_jarxoLCe-LvY-hWr_1769057779?w=1112&h=1170&type=image/png)  

3、 下载案例编辑器：下载完成后导出到一个自己方便管理的文件夹

💡 关键提示：Mac 电脑打开CodeBuddy报错，主要是目录权限问题，可以把报错信息发给元宝，然后根据提示操作就可以解决了

### 第一步：打开对话框

在 CodeBuddy 中打开 aicase-editor 文件夹，你会看到左侧对话框

![](https://wdoc-76491.picgzc.qpic.cn/MTY4ODg1MDUyOTQ5ODE3Nw_210811_8Ied21Q0cNUkvurC_1769058237?w=2400&h=1600&type=image/png)  

### 第二步：告诉 AI 你要做什么

方法一：拖动Skill.md到对话框，然后说明一下你的案例需求；

![](https://wdoc-76491.picgzc.qpic.cn/MTY4ODg1MDUyOTQ5ODE3Nw_3861_mGrl1dZupI7ZqWum_1769058299?w=2400&h=1600&type=image/png)  

方法二：复制下面这段话，把 \[...\] 部分换成你的真实数据，发在对话框里：

```
@Skill.md 请帮我制作一个广告投放案例，以下是数据： [直接粘贴你的投放数据表格，Excel/Adata表格直接复制粘贴即可] 客户信息：- 公司：[公司名称]- 产品：[产品名称]- 行业：[所属行业]
```

小技巧：案例需求比较多， 用对话框粘贴数据容易混乱？

1、 新建左侧资源管理器里，新建一个文件，命名为 「xxx.md」

2、 案例需求输入到这个「xxx.md」里面；

3、 拖动「xxx.md」和 「Skill.md」到对话框，直接告诉做一个案例即可；

![](https://wdoc-76491.picgzc.qpic.cn/MTY4ODg1MDUyOTQ5ODE3Nw_882766_UYicx5gFT6K7uUEv_1769058666?w=3046&h=2360&type=image/png)  

💡 关键提示：要求 AI制作案例时 开头的 @Skill.md 一定要加！这是告诉 AI "按专业流程来"的暗号

## 🎨 想微调？用可视化编辑器

生成的案例已经很好了，但如果你想改个标题、换个颜色，不用求 AI 重新生成，直接用编辑器改。

### 常用操作速查

| 我想… | 这样做 |
| :--- | :--- |
| 打开刚生成的案例 | 点击"开始编辑" → 「已生成案例」→ 选择文件 |
| 改文字 | 双击文字，直接敲键盘 |
| 换颜色 | 点击元素 → 右侧面板改颜色 |
| 换图片 | 双击图片 → 选择新图片 |
| 撤销误操作 | ⌘ + Z（Mac）/ Ctrl + Z（Windows） |
| 导出图片 | 点击右上角"导出" → 选择 JPG |

## 📁 文件在哪？

做完的案例都在 public/ 文件夹：

```
public/├── XX案例-数据亮点分析思路.md   ← AI 的分析过程（留档用）├── XX案例-最新版.md             ← 案例文档（文字版）├── XX案例-竖版.html             ← 长图版（发朋友圈）└── XX案例-横版.html             ← 演示版（放 PPT）
```

跟 AI 对话时，这些话术很好用：

| 场景 | 直接说 |

| 开始做案例 | @Skill.md 帮我做案例，数据如下：... |
| 改标题 | “标题改成 XXX” |
| 调整重点 | “把 ROI 作为第一亮点” |
| 隐藏某个数据 | “不要展示 CTR” |
| 只要横版 | “只生成横版” |
| 重新生成 | “重新生成 HTML” |

## ⚠️ 几个小提醒

1\. 记得加 @Skill.md - 这是让 AI 进入"案例制作模式"的开关

2\. 好好回答 AI 的问题 - AI 问的都是关键问题，乱答会影响效果

3\. 导出前检查数字 - 核对一下关键数据有没有写错

4\. 改完记得导出 - 编辑器里改的东西，要点"导出"才能保存成图片

## 遇到问题？

| 问题 | 解决方法 |
| :--- | :--- |
| AI 好像没按流程来 | 检查开头有没有加 @Skill.md |
| 生成的案例数据不对 | 回到对话，告诉 AI “XX数据写错了，应该是 XX” |
| 编辑器打不开 | 终端里先运行 npm install，再运行 npm run dev |
| 导出的图片模糊 | 导出时选择更高的分辨率 |
