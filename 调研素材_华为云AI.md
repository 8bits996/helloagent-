## 一、华为云AI产品矩阵

### 1.1 盘古大模型 5.5（HDC 2025发布）

#### Pangu NLP
- **718B参数深度思考模型**：256专家MoE架构
- **快慢思考切换**：简单问题敏捷回复，复杂问题深度推理，效率提升**8倍**
- **DeepDiver技术**：5分钟完成10步复杂问答，生成万字专业调研报告
- 增强知识推理、工具调用、数学能力

#### Pangu 多模态
- **Pangu世界模型**：生成数字物理空间，训练智能驾驶/具身智能
  - 输入驾驶场景→生成模拟摄像头视频+激光雷达点云
  - 广汽集团案例：像素级映射，模型迭代2天一版
- **CloudRobo具身智能平台**：数据合成→模型开发→仿真验证
- **R2C协议**：开放机器人云端协议

#### Pangu CV
- **30B参数CV模型**：业界最大，新MoE架构
- 泛视觉：图像、红外、激光雷达、光谱、雷达
- 中石油"昆仑大模型"：亚毫米级检测，效率提升40%

#### Pangu 预测
- 三元组Transformer统一架构：表格+时序+图像
- 案例：海螺水泥(原料配比)、宝武钢铁(日省20吨燃料)、中铝(年节电2600万kWh)

#### Pangu 科学计算
- 气象预报、能源预测

#### 五大行业垂类模型
医疗、金融、政府、工业、汽车（2025年6月上线）

### 1.2 AI基础设施 — CloudMatrix 384超节点
- 384个专用NPU + 192个鲲鹏CPU点对点互联
- 单卡推理吞吐：**2,300 tokens/s**（近4倍提升）
- 可级联至**160,000卡**超大规模集群
- 算训一体化："白天推理，晚上训练"

### 1.3 ModelArts Versatile（企业级AI Agent平台）
- 企业构建AI Agent的最佳平台
- 智能工具链：Agent生成时间从数天→数分钟
- **CodeArts Doer**：6个专用研发智能体，效率提升40%
  - ArchRAG + 动机遗忘机制
  - 模拟人类认知进行代码生成和自愈
- **GaussDB Doer**：AI数据库管理员

## 二、CodeArts 代码智能体

华为云自研AI编码产品，对标CodeBuddy/Copilot。基于盘古研发大模型。

### 2.2 核心功能
| 功能 | 说明 |
|------|------|
| 项目级代码生成 | 跨文件生成代码 |
| 智能续写 | 沉浸式编码心流 |
| 研发知识问答 | 编程问题+技术方案辅助 |
| 单元测试生成 | 自动生成+即时测试修复 |
| Agent知识增强 | 深度理解项目知识，精准检索 |

- **企业版**：待推出

## 五、关键启示

1. **多模型策略**：华为选择GLM+DeepSeek而非自研，降低研发成本
2. **行业深耕**：500+场景落地是华为的强壁垒
3. **信创优势**：政府/国企客户天然倾向华为
4. **CodeArts仍早期**：公测阶段，Agent能力和企业版尚未成熟，是CodeBuddy的窗口期
