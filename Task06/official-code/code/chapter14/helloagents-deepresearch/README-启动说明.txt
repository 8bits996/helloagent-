======================================================
  DeepResearch Agent - 启动说明
======================================================

【快速启动】
1. 双击运行: START.bat
2. 等待两个窗口打开 (不要关闭)
3. 按任意键，浏览器自动打开
4. 访问: http://localhost:5174

【停止服务】
1. 双击运行: STOP.bat
   或
2. 直接关闭两个服务窗口

【文件说明】
- START.bat  : 启动脚本 (推荐使用)
- STOP.bat   : 停止脚本
- 快速启动.bat : 备用启动脚本 (如果乱码用 START.bat)

【重要提示】
✓ 保持两个命令窗口打开，服务才会持续运行
✓ 可以最小化窗口，但不要关闭
✓ 如果看到乱码，请使用 START.bat

【服务地址】
- 前端界面: http://localhost:5174
- 后端API:  http://localhost:8000
- API文档:  http://localhost:8000/docs
- 健康检查: http://localhost:8000/healthz

【常见问题】
Q: 双击后窗口一闪而过？
A: 确保已安装 Python 和 Node.js

Q: 提示端口被占用？
A: 运行 STOP.bat 停止现有服务

Q: 服务无法访问？
A: 检查两个命令窗口是否还在运行
