## 一、阿里巴巴 AI 战略投入

### 1.1 3800亿投资计划

2025年2月，阿里巴巴CEO吴泳铭宣布**未来三年投入超过3800亿元人民币**（约530亿美元）用于建设云和AI硬件基础设施。

- 创下中国民营企业在云和AI硬件基础设施投资的最高纪录
- 总额超过阿里过去十年投入总和
- 核心方向：GPU算力建设、自研AI芯片（平头哥PPU）、开源模型

### 1.2 核心战略方向

1. **AI基础设施**：大规模GPU算力建设、自研AI芯片
2. **开源领先**：Qwen系列全面开源，争夺全球开源模型第一
3. **AI云增长**：阿里云2026财年Q2营收398.24亿元，同比增长34%
4. **全栈垂直整合**：从芯片→模型→平台→应用的全链路

## 三、开源生态

### 3.1 Qwen3系列

- 2025年4月发布，首个国内"混合推理"开源模型
- 登顶全球最强开源模型（多项评测领先）
- Qwen3.5为最新旗舰版本

### 3.2 Qwen3-Coder

- 2025年7月开源
- AI编程性能达开源最强水平
- 支持多种编程语言

### 3.3 Qwen Code

- 基于Gemini CLI改造的开源命令行AI工具
- 专为Qwen3-Coder优化
- 支持MCP Server

### 3.4 AgentScope

- 多Agent协作开源框架
- 阿里达摩院出品

### 3.5 ModelScope（魔搭）

- 全球最大中文开源模型社区之一
- 3000+MCP服务
- 数千个开源模型
