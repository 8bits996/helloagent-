# 🚀 系统启动说明

## 📋 前置要求

确保以下服务已安装：
- ✅ Python 3.10+
- ✅ 所有依赖已安装 (`pip install -r requirements.txt`)
- ✅ 配置好 `.env` 文件

---

## 🎯 完整启动流程

### 方式一：使用启动脚本（推荐）

**Windows 用户：**
```bash
# 双击运行
start_services.bat
```

### 方式二：手动启动

**步骤1: 启动 FastAPI 后端**

打开终端1，运行：
```bash
cd /path/to/contract-review-ai
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**步骤2: 启动 Streamlit 前端**

打开终端2，运行：
```bash
cd /path/to/contract-review-ai
python -m streamlit run app/frontend.py --server.port 8501
```

## 🔧 环境变量配置

编辑 `.env` 文件：

```env
# LLM 配置
LLM_PROVIDER=siliconflow
LLM_API_KEY=your_api_key_here
LLM_MODEL=deepseek-ai/DeepSeek-V3
LLM_BASE_URL=https://api.siliconflow.cn/v1

# 评审模式
REVIEW_MODE=api  # 可选: api, mock, cli
```

## ❓ 常见问题

### 端口被占用

```bash
# 查找占用端口的进程
netstat -ano | findstr :8000

# 结束进程
taskkill /PID <进程ID> /F
```

### LLM 调用失败

1. 检查 API Key 是否正确
2. 检查网络连接
3. 查看后端日志获取详细错误信息

### 知识库加载失败

确保知识库 CSV 文件：
- 编码为 UTF-8
- 放置在 `知识库/` 目录下
- 文件格式正确
