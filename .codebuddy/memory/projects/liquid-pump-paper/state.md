## 研究对象
- 某型液冷泵：0.5MPa / 3000L/h / 4500rpm / 2.2kW直流无刷电机
- 试验依据：GJB 4058-2000

## 核心结论
- 优化前：振动加速度总级 107.1/107.8 dB（裕度仅2.2~2.9 dB）
- 优化后（k=2200 N/mm, ζ=0.18）：100.6/101.5 dB（裕度提升至8~9 dB）
- 降低量：6.3~6.5 dB
