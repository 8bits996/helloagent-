🏃 东莞市体育中考备考完整方案

基于《东莞市初中学业水平考试体育与健康科目考试实施方案（2025年修订）》

📅 当前时间：2025年12月7日 🎓 当前年级：初二上学期 🎯 中考时间：2027年4-5月 💪 个人优势：长跑（800米/1000米）
📋

政策改革解读

发布时间：2025年12月3日，东莞市教育局正式发布《东莞市初中学业水平考试体育与健康科目考试实施方案（2025年修订）》。

✅ 适用于你的新政策（2026年、2027年中考）

2026年中考起

1. 成绩计算方式优化 - 三项互补

必考、选考、体能素质三项考试单项最高分为120分，三项成绩可互补。平均分超过100分按100分计，未超过按实际得分折算。

💡 对你的意义

长跑强项可冲120分，即使其他两项各80分，平均分也有93.3分！强项能有效弥补弱项。

2026年中考起

2. 体能素质项目改为自选（五选一）

体能素质考试项目从之前由市教育局统一抽取改为考生自选。

可选项目：

  • ①引体向上（男）、一分钟仰卧起坐（女）
  • ②一分钟双杠杠端跳起支撑前摆下
  • ③立定跳远
  • ④原地双手正面投掷实心球
  • ⑤平行梯悬垂攀移（仅2026-2027年有）
2026年中考起

3. 选考项目范围扩大（九选一）

选考项目从5选1扩大为9选1：

  • ①篮球
  • ②足球
  • ③排球
  • ④气排球
  • ⑤素质四项
  • ⑥十字障碍跑
  • ⑦三角蛇形跑
  • ⑧60米跨栏
  • ⑨上步三级跳远
2026年中考起

4. 部分项目难度降低

引体向上（男）、平行梯悬垂攀移（女）、篮球、足球、排球等项目评分标准适当下调。

❌ 不适用于你的政策（2028年中考起）

2028年中考起

平时考核分值提高至30分（50%）

平时考核从6分（10%）提高至30分（50%），初一、初二、初三各10分。

你的情况：仍按原方案，平时考核6分（占10%）。

2028年中考起

新增"三小球"项目

选考项目增加乒乓球、羽毛球、网球。

你的情况：虽然你会打羽毛球和乒乓球，但2027年中考还未纳入这些项目，无法选择。

2028年中考起

体能素质调整为四选一

取消平行梯悬垂攀移，保留其他四项。

三项互补计算示例

📐 计算公式

三项单项最高分均为120分，最终得分 = (必考分 + 选考分 + 体能素质分) ÷ 3

如果平均分 ≥ 100分，按100分计；如果 < 100分，按实际得分折算为54分满分。

《国家学生体质健康标准》测试（平时考核6分）

📌 重要说明

平时考核6分来源于每学年的体质健康测试，达到及格标准即可获得该学年满分。测试依据《国家学生体质健康标准（2014年修订）》执行。

测试时间安排

学年测试时间分值状态
初一（七年级）2024年9月-12月2分已完成
初二（八年级）2025年9月-12月2分约9-12个月后
初三（九年级）2026年9月-12月2分待完成
⏰ 测试时间说明

根据教育部要求，各学校每学年9月1日至12月31日期间完成体质健康测试并上报数据。大多数学校安排在10月-11月进行集中测试。具体时间以学校通知为准。

全部考试项目详解

以下详细分析2026-2027年东莞体育中考所有可选项目的考试内容、难度、特点、训练要点，供选择参考。

一、必考选考类（2选1）

🏃长跑：1000米(男)/800米(女) 推荐

项目难度训练周期稳定性你的情况
中长跑⭐⭐⭐长期积累高强项
📋 评分标准（部分）
分值男生1000米女生800米
120分3'20"3'05"
100分3'40"3'25"
80分4'05"3'55"

🔑 技术要点
  1. 起跑：站立式起跑，听到信号后快速启动但不要冲太快
  2. 途中跑：保持匀速，呼吸节奏与步频配合（两步一呼两步一吸）
  3. 冲刺：最后200米加速，最后50米全力冲刺
  4. 配速策略：前半程稍慢，后半程加速，避免前快后慢

二、体能素质类（5选1）

🎯原地双手正面投掷实心球 推荐

项目难度训练周期稳定性你的情况
实心球⭐⭐2-3个月中高推荐
🔑 技术要点
  1. 握球：双手五指张开，包住球的两侧和后下方
  2. 预摆：球从胸前向后上方引，身体后仰形成"满弓"
  3. 发力顺序：蹬地→转髋→挺胸→甩臂→拨指
  4. 出手角度：约38-42度，太高或太低都会影响远度
✅ 优势
  • 技术性项目，掌握正确方法后提升快
  • 核心力量训练对长跑也有帮助
  • 不需要爆发力，适合耐力型选手

🦘立定跳远 不推荐

项目难度训练周期稳定性你的情况
立定跳远⭐⭐⭐3-4个月中不擅长
🔑 技术要点（如需训练）
  1. 预摆：双臂前后摆动2-3次，配合屈膝
  2. 起跳：蹬地、摆臂、展体同时进行，起跳角度约45度
  3. 腾空：收腹举腿，保持身体平衡
  4. 落地：小腿前伸，脚跟先着地，屈膝缓冲
⚠️ 难点

需要较好的爆发力和协调性，长跑型选手通常不占优势。考试时容易受紧张情绪影响，发挥不稳定。

💪引体向上（男生）

项目难度训练周期稳定性适合人群
引体向上⭐⭐⭐⭐4-6个月中上肢力量好、体重轻
🔑 技术要点
  1. 握杠：正握（手心向前），握距略宽于肩
  2. 引体：背部和手臂同时发力，下巴超过杠面
  3. 下放：控制速度下放，不要完全放松
  4. 呼吸：上拉时呼气，下放时吸气
⚠️ 难点

对上肢力量要求高，体重较大的同学较难。2026年起评分标准已下调，但仍是公认最难的项目之一。

🏋️一分钟仰卧起坐（女生）

项目难度训练周期稳定性适合人群
仰卧起坐⭐⭐1-2个月高女生首选
🔑 技术要点
  1. 准备姿势：仰卧，双手抱头，双腿屈膝90度
  2. 起身：腹部发力，肘部触及膝盖
  3. 还原：肩胛骨触垫即可，不需要完全躺平
  4. 节奏：保持稳定节奏，不要前快后慢
✅ 优势

技术简单，训练周期短，女生体能素质首选项目。

🤸一分钟双杠杠端跳起支撑前摆下

项目难度训练周期稳定性适合人群
双杠支撑⭐⭐⭐3-4个月中上肢力量好
⚠️ 注意

需要专业器械和指导，动作有一定技术要求。建议在体育老师指导下训练。

🐒平行梯悬垂攀移（仅2026-2027年）

项目难度训练周期稳定性适合人群
平行梯攀移⭐⭐⭐3-4个月中握力好、体重轻
⚠️ 注意

2028年起取消此项目。对握力和上肢耐力要求较高，训练时容易磨破手掌。

三、选考类（9选1）

🏐排球 / 气排球 推荐

项目难度训练周期稳定性你的情况
排球/气排球⭐⭐3-4个月高推荐

考试内容：拦网—传球—传球接击球—垫球—垫球接击球—发球（运动能力组合）

📋 评分标准
分值完成次数
120分12次
100分10次
80分8次
60分6次
🔑 技术要点
  1. 垫球：两手交叉叠放，大拇指并拢，用前臂平面击球，腿部蹬地发力
  2. 传球：双手呈半球形，手指触球，利用手指和手腕弹力将球传出
  3. 发球：下手发球最稳定，击球点在腰部高度，手掌根部击球
  4. 拦网：起跳时机、手型和落地姿势
✅ 为什么推荐
  • 2026年起评分标准已下调
  • 技术相对简单，短期可提升
  • 考试稳定性高，不易失误
  • 气排球比排球更轻更软，更容易控制

  3. 加速：转向后迅速加速

  2. 转向时重心下降，快速变向
  3. 保持稳定节奏，避免失误

🏋️素质四项 可考虑

项目难度训练周期稳定性适合人群
素质四项⭐⭐⭐3-4个月中综合素质好

考试内容：连续完成俯卧撑8次(男)/6次(女)—跳跃5个障碍—连续前滚翻2次—30米跑

🔑 技术要点
  1. 俯卧撑：标准姿势，胸部接近地面
  2. 跳跃障碍：连续跳跃，节奏稳定
  3. 前滚翻：低头团身，保护颈部
  4. 30米跑：全力冲刺
⚠️ 注意

需要综合能力，前滚翻需要专业指导。如果某一项特别弱，会拖累整体成绩。
