## 🏗️ 架构核心决策

1. **三层管控**：PM → Orchestrator → GateKeeper（每阶段质量门控）
2. **五阶段通用流程**：采集 → 分析 → 洞察 → 验证 → 交付
3. **贯穿型智能体**：TheoryAgent + ExplorerAgent + MemoryAgent（全程参与）
4. **双向反馈循环**：验证失败可回溯至任意前序阶段
5. **通用骨架 + 场景映射**：一套基础架构适配多场景
