> 调研日期：2026年3月27日

## 一、产品定位

AgentKit是火山引擎的企业级AI Agent开发平台，采用Serverless托管模式，为开发者提供Agent的**构建、部署和运维**全套解决方案。

- **定位**：企业级AI Agent开发框架
- **部署模式**：Serverless托管
- **目标用户**：企业开发者、Agent开发团队

---

## 二、八大核心模块

| 模块 | 功能 | 详情 |
|------|------|------|
| **Agent Core** | Agent核心引擎 | 支持多框架兼容，核心推理和决策引擎 |
| **MCP 生态** | 外部工具连接 | 支持MCP协议，连接外部工具和服务 |
| **长短记忆管理** | Agent记忆持久化 | 对话历史、知识记忆的存储和管理 |
| **知识库** | 企业知识管理 | 企业知识的向量化存储与检索（RAG） |
| **应用观测** | 运行监控 | 全链路Agent运行监控和可视化 |
| **评测系统** | 效果评估 | Agent效果的定量评估 |
| **安全防护** | 安全保障 | 大模型防火墙、内容安全 |
| **部署运维** | 基础设施 | Serverless托管、弹性伸缩 |

---

## 三、技术特点

1. **基于字节跳动实战经验**：基于字节跳动大量AI应用研发经验构建
2. **开发快、部署稳**：全链路支持，从开发到企业级平台部署
3. **多框架兼容**：可扩展，支持多种Agent框架
4. **低代码量**：智能会议助手案例仅需不足70行代码
5. **Python SDK已开源**：开发者可快速接入

---

## 四、在火山Agent矩阵中的位置

```
┌──────────────────────────────────────────────────────────┐
│                    AgentKit (底层框架)                      │  ← 本产品
│         企业级Agent构建、部署、运行基础设施                    │
├────────────────────────┬─────────────────────────────────┤
│      扣子 (Coze)        │          HiAgent                │
│  面向开发者/中小企业      │     面向大型企业                  │
└────────────────────────┴─────────────────────────────────┘
```

AgentKit是整个Agent产品矩阵的**底层基础设施**，Coze和HiAgent都基于AgentKit的能力构建。
