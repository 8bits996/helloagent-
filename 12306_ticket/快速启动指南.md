# 12306 抢票快速启动指南

## 🚀 快速启动步骤（3分钟完成）

### 第1步：准备页面（1分钟）
1. 打开浏览器，访问 https://www.12306.cn/
2. 登录你的账号（用户名：陈卓）
3. 点击"车票预订"
4. 设置查询条件：
   - 出发地：襄阳
   - 目的地：武当山
   - 日期：**2026-02-09**
   - 点击"查询"按钮

### 第2步：打开控制台（30秒）
1. 在页面上按 **F12** 键
2. 点击顶部 **"Console"**（控制台）标签
3. 确认看到一个光标在闪烁

### 第3步：执行脚本（30秒）
1. 打开文件：`auto_grab_simple.js`
2. 全选内容（Ctrl + A）
3. 复制（Ctrl + C）
4. 粘贴到浏览器控制台（Ctrl + V）
5. 按回车键执行

### 第4步：启动抢票（10秒）
在控制台输入以下命令之一：

**方式1：定时抢票（推荐）**
```javascript
AutoGrab.schedule()
```

**方式2：立即抢票（已过16:00时使用）**
```javascript
AutoGrab.monitor()
```

**停止抢票：**
```javascript
AutoGrab.stop()
```

## 🆘 故障排除

### 问题1：脚本未执行
**解决**：
- 确认在"Console"标签页
- 检查是否有错误信息（红色）
- 刷新页面重新粘贴脚本

### 问题2：显示"未找到查询按钮"
**解决**：
- 确保在购票查询页面
- 点击"查询"按钮看到车次列表
- 刷新页面重新执行脚本

### 问题3：乘客选择失败
**解决**：
- 确认4位乘客都在常用联系人中
- 刷新页面重新执行脚本
- 手动选择乘客后点击提交

### 问题4：长时间未抢到票
**解决**：
- 检查起售时间是否正确
- 确认电脑时间准确
- 查看控制台日志了解进度
- 考虑修改起售时间后重试
