## 一、产品概述

Cursor 是基于VS Code fork的AI原生IDE，2026年用户超百万，是代理式IDE品类的领导者。2026年发布Cursor 2.0，推出首个自研大编程模型，解决"无护城河"问题。

### Credits机制
- 2025年改为信用额度制，按实际AI计算资源消耗扣减
- **Auto模式**（自动选最经济模型）几乎不消耗credits
- 手动选高端模型（Claude Sonnet、GPT-4o等）消耗快
- 额度用完后**不停止服务**，切换到慢速排队模式，Tab补全和Auto模式不受影响

## 三、核心功能

### 3.1 Composer（多文件编辑）
- 业界公认的多文件编辑体验最佳
- 支持跨文件理解依赖关系后统一修改
- 支持自然语言描述需求后生成/修改多文件代码

### 3.2 Background Agents（后台代理）
- 在云端独立VM中自动执行任务
- 可克隆仓库、运行测试、推送代码
- Pro+及以上解锁
- 与本地编辑互不干扰

### 3.3 多模型支持
- GPT-5、Claude系列、Gemini等
- Auto模式自动选择最经济模型
- 用户可手动切换模型

### 3.4 代码库索引
- 自动索引项目代码
- 基于上下文的精准代码生成

## 四、Cursor 2.0 重大更新

- **自研大编程模型**：首次推出自有模型，不再完全依赖第三方
- 解决长期"护城河"问题（此前被批评只是Claude/GPT的壳）
- 具体模型性能待进一步验证

## 五、优缺点分析

### 优势
- 体验最polished，UI/UX业界领先
- 社区最大（百万用户），资源和模板丰富
- 多模型灵活切换
- Background Agents实用性强

### 劣势
- Credits定价有时不可预测
- 部分VS Code扩展不兼容
- 顶配$200/月价格昂贵
- 企业版($40/人)比Copilot Business($19/人)贵一倍
