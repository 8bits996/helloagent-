## 一、产品概述

阿里云百炼是**一站式大模型开发及应用构建平台**，聚合了通义千问（Qwen）全系列等百余款大模型。

- **定位**：一站式大模型服务平台（PaaS）
- **官网**：https://bailian.console.aliyun.com/
- **目标**：实现"零门槛"落地AI应用，5分钟即可开发一款大模型应用

---

## 二、核心能力

### 2.1 丰富的模型矩阵

| 模型 | 定位 | 特点 |
|------|------|------|
| Qwen3.5-Max | 旗舰全能 | 原生多模态，最强性能 |
| Qwen3.5-Plus | 企业应用 | 性价比高 |
| Qwen-Turbo | 高并发 | 低成本 |

| Qwen-VL | 视觉专用 | 图像理解 |
| 万相 2.6 | 视频生成 | 15秒超写实视频 |
| GLM | 第三方 | 智谱模型 |
| Kimi | 第三方 | 月之暗面模型 |

### 2.2 Agent开发能力

| 能力 | 详情 |
|------|------|
| 低代码开发 | 拖拽式工作流编排，100+行业模板 |
| 高代码开发 | Python SDK、MCP协议 |
| MCP Server托管 | 一键注册托管自定义工具 |
| Agent Store | 开发者上架智能体变现 |
| MCP市集 | 生态伙伴工具插件 |
| 智能体应用 | 零代码Agent构建 |
| 工作流编排 | 可视化流程设计 |
| 知识库 | RAG知识检索增强 |

### 2.3 安全与合规

- **数据安全**："只计算不留存"模式，推理后数据自动销毁
- **加密**：全链路TLS加密
- **合规**：等保三级、安全白皮书2.0标准
- **适用场景**：金融、政务合规

### 2.4 商业生态

- **Agent Store**：智能体商店，支持开发者上架变现
- **支付宝AI付**：集成支付能力，可快速实现商业闭环
- **MCP市集**：工具插件生态
